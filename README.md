# Bubble Sort in Assembly (emu8086) 💻

A low-level implementation of the **Bubble Sort** algorithm written in **8086 Assembly Language**. This project demonstrates register management, memory addressing, and manual loop control.

## 🛠️ Features
- **Algorithm:** Bubble Sort (Ascending Order).
- **Environment:** Developed and tested on **emu8086**.
- **Memory Management:** Direct manipulation of data segments and pointers.

## 📂 File Structure
- `bubblesort.asm`: The source code containing the sorting logic.

## 🚀 How to Run
1. Download and install **emu8086**.
2. Open `bubblesort.asm` in the emulator.
3. Click **Emulate** and then **Run**.
4. View the sorted array in the **Memory** or **Variables** window.
